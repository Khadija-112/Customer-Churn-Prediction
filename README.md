# Customer-Churn-Prediction
An end-to-end Customer Churn Prediction project that combines Machine Learning and Power BI to identify customers at risk of churn, uncover key business drivers, and support data-driven retention strategies.

## 📖 Overview

This project analyzes customer behavior using historical telecom customer data to predict churn and visualize actionable insights. It includes data cleaning, feature engineering, predictive modeling, and an interactive Power BI dashboard that enables stakeholders to monitor churn trends and make informed business decisions.

## ✨ Features

- Customer churn prediction using Machine Learning
- Interactive Power BI dashboard
- Customer segmentation by age, tenure, contract, and payment method
- Churn rate analysis across multiple customer groups
- Key Influencers analysis to identify churn drivers
- Dynamic slicers for interactive filtering
- Business-focused KPIs and visual insights

## 📈 Key Insights

- Overall customer churn rate is **26.5%**.
- Customers using **Electronic Check** have the highest likelihood of churning.
- **Month-to-Month** contracts experience significantly higher churn than long-term contracts.
- Customers with **Fiber Optic** internet service show higher churn compared to DSL users.
- Customer tenure strongly influences churn, with newer customers more likely to leave.
- Contract type and payment method are among the strongest churn predictors.
